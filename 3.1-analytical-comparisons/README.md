To run one of the analytical cases

`> python3  internalvariable_viscoelastic_freesurface.py --case case`

where `case` is one of

- "elastic-compressible"
- "elastic-incompressible"
- "viscoelastic-compressible"
- "viscoelastic-incompressible"

Outputs of vertical displacement at the surface at each timestep and
analytical error convergence for the four cases are provided in the
respective folders.
