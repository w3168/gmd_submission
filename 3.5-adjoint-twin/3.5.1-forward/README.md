The script `forward.py` can be used to reproduce the forward 2d cylinder runs
of Section 3.5.1. 

To run the laterally varying viscosity case

`python forward.py`

To run the 1D viscosity case 

`python forward.py --radial_visc`

Surface displacement and velocity outputs can be found in the outputs folder
including the checkpoint file for the laterally varying case used as synthetic
observations as part of the inversion experiments in Sections 3.5.2 - 3.5.4.
