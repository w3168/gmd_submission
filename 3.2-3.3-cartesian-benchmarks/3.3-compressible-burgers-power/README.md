This folder contains surface displacement outputs for each case in Section 3.3.

`long-burgers0.1`: Long simulation, compressible, Burgers rheology (viscosity 2 / viscosity 1 = 0.1)

`long-burgers0.5`: Long simulation, compressible, Burgers rheology (viscosity 2 / viscosity 1 = 0.5)

`long-compressible`: Long simulation, compressible, Maxwell rheology

`long-power`: Long simulation, compressible, composite power-law rheology

`short-burgers0.1`: Short simulation, compressible, Burgers rheology (viscosity 2 / viscosity 1 = 0.1)

`short-burgers0.5`: Short simulation, compressible, Burgers rheology (viscosity 2 / viscosity 1 = 0.5)

`short-compressible`: Short simulation, compressible, Maxwell rheology

`short-power`: Short simulation, compressible, composite power-law rheology
