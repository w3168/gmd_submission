This folder contains surface displacement outputs for each case in Section 3.2.

`long-1d`: Long simulation, incompressible, Maxwell rheology, 1D viscosity profile

`long-lvv`: Long simulation, incompressible, Maxwell rheology, 3D viscosity field

`short-1d`: Short simulation, incompressible, Maxwell rheology, 1D viscosity profile

`short-lvv`: Short simulation, incompressible, Maxwell rheology, 3D viscosity field
